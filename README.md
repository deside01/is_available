# Дата выполнения: 16.11.2025
## Сервис для определения статуса интернет-ресурсов
Данные сохраняются в data.json, json был выбран с целью быстро поиска данных.
Данные сохраняются синхронно, поэтому одновременная запись для нескольких запросов недоступна. Поэтому лучше использовать базу данных.
Есть Graceful Shutdwon

## Как запустить
1. Скопировать репозиторий
```
git clone https://github.com/deside01/is_available.git
```
2.  Переименовать .env.example в .env
3. Запустить run.bat (Windows)
